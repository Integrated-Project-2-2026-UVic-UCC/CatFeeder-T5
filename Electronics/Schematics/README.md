# Schematics

This folder contains the electrical schematics of the Automatic Cat Feeder project.

The schematics define how the electronic components are connected, including the microcontroller, sensors, actuators, power supply, protection elements, and communication signals.

## Purpose

The purpose of this folder is to:

- Document the electrical design of the project.
- Show how each electronic component is connected.
- Support wiring, debugging, and testing.
- Keep editable schematic files and exported versions organized.
- Track changes between schematic revisions.
- Help detect electrical design errors before assembly.
