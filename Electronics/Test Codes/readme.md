# Test Codes

This folder contains test programs used to validate the electronic components of the Automatic Cat Feeder project.

Each test is focused on checking one specific component or one small part of the system before integrating everything into the final firmware.

## Purpose

The purpose of these test codes is to:

- Verify that each component works correctly.
- Check the wiring and pin assignments.
- Validate sensor readings.
- Test actuator movement.
- Detect electrical or connection issues before full integration.
- Document the expected behaviour of each component.
