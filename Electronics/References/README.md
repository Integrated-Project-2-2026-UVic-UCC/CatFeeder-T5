
# References

This folder contains external references used during the development of the electronics part of the Automatic Cat Feeder project.

The purpose of this folder is to keep track of useful technical sources, official documentation, tutorials, articles, standards, and design references that helped support design decisions.

## Purpose

The references stored here are used to:

- Justify technical decisions.
- Support component selection.
- Document external sources used during development.
- Keep useful links organized.
- Avoid losing important technical information.
- Distinguish project documentation from external information.
