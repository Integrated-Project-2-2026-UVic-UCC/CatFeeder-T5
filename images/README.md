
# Images

This folder contains the images used in the main `README.md` file of the **Automatic Cat Feeder** project.

The images are mainly used to document the electronic components, mechanical parts, wiring, assembly process, and final prototype.

---

## Folder purpose

The purpose of this folder is to keep all project images organized in one place, instead of linking external images directly inside the README.

This makes the documentation easier to maintain and more reliable when the project is uploaded to GitHub.
